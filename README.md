# 🎨 Portfólio — Valdeise Lemos

> Portfólio pessoal de Valdeise Lemos, designer com foco em ilustração, embalagens e identidade visual. Desenvolvido por **Patricia de Oliveira** e **João Victor Pereira**.

🔗 **[Ver projeto ao vivo](https://patricia-de-oliveira.github.io/portifolio_valdeise_patricia_joao-victor/)**

---

## 📋 Sobre o Projeto

Este projeto é um portfólio digital desenvolvido para apresentar o trabalho e as competências de **Valdeise Lemos**, designer especializada em ilustração digital, design de embalagens e identidade visual. O site foi construído com HTML, CSS e JavaScript puros, com foco em uma experiência visual limpa e responsiva.

---

## 🖥️ Páginas

| Página | Descrição |
|--------|-----------|
| `index.html` | Página principal com apresentação, habilidades, softwares, soft skills e contatos |
| `projetos.html` | Galeria de projetos da designer |

---

## ✨ Funcionalidades

- Carrossel de fotos na seção de apresentação
- Listagem de habilidades técnicas e soft skills
- Seção de softwares utilizados (Illustrator, Photoshop, Figma, Procreate)
- Links diretos para redes sociais e contato (Behance, LinkedIn, Instagram, e-mail)
- Design responsivo para diferentes tamanhos de tela
- Navegação entre as páginas Sobre e Projetos

---

## 🛠️ Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript** (vanilla)
- **GitHub Pages** — hospedagem do projeto

## 🚀 Como Executar Localmente

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/victornbjj/portifolio_valdeise_patricia_joao-victor.git
   ```

2. **Acesse a pasta do projeto:**
   ```bash
   cd portifolio_valdeise_patricia_joao-victor
   ```

3. **Abra o arquivo `index.html` no seu navegador** — nenhuma instalação adicional é necessária, pois o projeto utiliza apenas HTML, CSS e JavaScript puros.
